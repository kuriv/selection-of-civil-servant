# 公务员遴选

> 公务员遴选是指党群机关、政府部门根据相关政策的规定，从已经具备公务员身份和公务员相关资历的人员中选出符合组织或相关部门要求的部门工作人员，是党政部门选拔人才的重要途径和方法。
